# Python Object-Oriented Programming

This repository is dedicated to studying Object-Oriented Programming (OOP) in Python through courses offered by Alura.

The main reference for this study is the course **“Python: avance na Orientação a Objetos e consuma API”**, in which it is possible to:

* Implement inheritance and abstract classes
* Understand and apply polymorphism
* Integrate Python projects with external applications
* Create JSON files in a practical way
* Create and activate virtual environments in Python
